# Tour Finder - iOS приложение для подбора туров

Крутое нативное iOS приложение для поиска и бронирования туров с личным кабинетом, созданное на SwiftUI.

## 🚀 Возможности

- 🔍 **Поиск туров** - Удобный поиск по странам и городам
- ❤️ **Избранное** - Сохраняйте понравившиеся туры (данные сохраняются локально)
- 👤 **Личный кабинет** - Управление профилем и статистикой
- 📱 **Современный UI** - Элегантный и минималистичный дизайн на SwiftUI
- 🎨 **Категории** - Фильтрация туров по типам отдыха
- 📋 **Детальная информация** - Полное описание каждого тура

## 🛠 Технологии

- **SwiftUI** - Современный фреймворк для UI
- **Swift** - Язык программирования
- **MVVM** - Архитектурный паттерн
- **UserDefaults** - Локальное хранение данных

## 🎨 Дизайн

Приложение использует современную цветовую палитру с акцентом на синий цвет и элегантную типографику. Все компоненты следуют принципам Human Interface Guidelines от Apple.

## 📝 Особенности реализации

- ✅ Нативная навигация с TabView
- ✅ Локальное сохранение избранного
- ✅ Фильтрация по категориям
- ✅ Поиск в реальном времени
- ✅ Адаптивная верстка
- ✅ Красивые градиенты и тени
- ✅ 6 готовых туров с данными

## 🚀 Запуск

1. Откройте проект в Xcode:
   ```
   Prodayo more.xcodeproj
   ```

2. Выберите симулятор или устройство

3. Нажмите ⌘R для запуска
